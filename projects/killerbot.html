KillerBot

2019 - 2022

GitHub Repo Trailer Top.gg Page

The Idea

A full-featured Discord bot designed from scratch to handle everything from server management to fun minigames — and eventually scaled to over 250,000 users across 600+ servers.

What I Built

  • Wrote the entire backend in C# using Discord.NET from scratch
  • Implemented full economy system with currency, shops, balances and transactions
  • Created advanced moderation suite, auto-moderation filters, and admin tools
  • Built entertainment features including trivia, games, weather and YouTube APIs
  • Handled high-volume data storage using JSON-based custom persistence layers

Achievements

  • Organically grew to serve over 250,000 Discord users across 600+ servers
  • Maintained 5-star Top.gg rating and highly positive user reviews
  • Integrated multiple public APIs to extend real-world data interaction
  • Delivered fully stable 24/7 uptime for years with constant updates & feature additions

Why This Was Fun

Every feature I added instantly touched thousands of users — watching people engage live with something I built was addictive.

Collaboration

Sole developer and maintainer. Designed, coded, maintained, and grew KillerBot entirely independently.
