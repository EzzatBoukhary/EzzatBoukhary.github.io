Campus Critters

Nov 2024

GitHub Repo

The Idea

Campus Critters makes spotting wildlife around UCF fun and easy. Students can log sightings, share images, and browse what others have found — directly from both web and mobile apps.

What I Built

  • Designed and built full-stack MERN web app with React (TypeScript) frontend and Node.js backend
  • Created secure authentication with email verification, JWT and hashed passwords
  • Implemented image uploads, tagging, filtering, and live location mapping via Leaflet
  • Developed Flutter mobile companion app synced with full backend

Achievements

  • Delivered both web & mobile apps fully functional across platforms
  • Architected scalable REST API for data consistency across clients
  • Integrated real-time dynamic filtering and category-based search
  • Handled full deployment pipeline using DigitalOcean cloud
  • Self-directed end-to-end product ownership and iteration

Why This Was Fun

Seeing real user reports show up on a live map as people interacted with the app was incredibly satisfying — plus I discovered animals I never knew were around UCF.
