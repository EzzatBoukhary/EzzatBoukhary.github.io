College Event Platform

March 2025 - April 2025

GitHub Repo Demo Video Final Report

The Idea

A full-stack platform that lets universities easily organize public, private, and RSO events — while enforcing real-world constraints like role-based permissions, approvals, and non-overlapping scheduling.

What I Built

  • Engineered full backend API from scratch with Node.js, Express and MySQL
  • Designed complete 3NF database schema with foreign keys, triggers, stored procedures and relational integrity
  • Built secure role-based access control for students, admins, super-admins
  • Implemented event approval pipelines, time conflict validation, and RSO member count triggers directly at the database layer
  • Developed frontend features in React with TypeScript for event creation, search, filtering, and commenting

Achievements

  • Led full-stack system architecture from broken initial state into production-ready design
  • Implemented complex constraint logic entirely at database level for maximum integrity
  • Built fully functional search, filtering and lookup endpoints for admin dashboards
  • Handled end-to-end API refactoring, routing, security, and error handling
  • Delivered polished final report, demo and functional application as lead backend developer

Why This Was Fun

There's something satisfying about translating real-world business rules into actual system constraints — and seeing everything work together like clockwork was incredibly rewarding.

Collaboration

Team of 3. I led backend design, data modeling, API architecture, complex triggers, and core frontend features while collaborating on integration and final delivery.
